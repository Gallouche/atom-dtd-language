# Language-dtd package for Atom

Adds syntax highlighting and snippets to DTD (Document Type Definition) files in Atom.

## Snippets

| Snippets      | Result                                   |
| ------------- |:----------------------------------------:|
| `<`           | <!NAME element>                          |
| `element`     | <!ELEMENT name (content)>                |
| `attlist`     | <!ATTLIST element attribute TYPE #VALUE> |
